## Key Features

### 1. Toggle Default Products
- **Location:** Top of My Shop page
- **Effect:** Show/hide default products in store
- **Note:** Can still edit markup when disabled

### 2. Add Custom Product
- **Button:** "Add Custom Product" (green button)
- **Fields:** Name, Network, Size, Base Price, Markup %
- **Auto-calculated:** Customer Price, Commission

### 3. Edit Markup
- **Button:** "Edit Price" on each product card
- **Change:** Markup percentage
- **Auto-updated:** Customer price and commission

### 4. Delete Product
- **Button:** "Delete" (red button, custom products only)
- **Note:** Default products cannot be deleted

## Troubleshooting

**Q: Products not showing?**
A: Check if default products are disabled (toggle switch)

**Q: Can't delete default products?**
A: Default products can only be disabled, not deleted

**Q: Commission not calculating?**
A: Ensure base price and markup are filled correctly

**Q: Need different markup?**
A: Create custom product with desired markup
